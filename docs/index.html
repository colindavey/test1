Keyboard tips: Check right menu for editing keyboard shortcuts. For navigation, use the arrow keys or:

  • j to move down.
  • shift+j to jump down.
  • k to move up.
  • shift+k to jump up.

If the item you are jumping to (for up) or from (for down) has children, jump jumps over the children, otherwise it jumps to the first (up) or last (down) sibling.
